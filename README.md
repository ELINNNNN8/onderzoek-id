# ONDERZOEK.ID

Prototype website interaktif untuk GitHub Pages.

## Isi fitur
- Landing page
- Peta interaktif Surabaya, Malang, Madura
- Tiga sudut pandang
- Timeline sejarah
- Kartu fakta media sosial
- Viewer arsip dummy

## Cara pakai
1. Upload `index.html` ke repository GitHub.
2. Aktifkan GitHub Pages.
3. Buka link situs yang muncul.
